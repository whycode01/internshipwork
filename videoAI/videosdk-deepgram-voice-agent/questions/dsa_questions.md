# Data Structures and Algorithms Questions

This file contains focused DSA questions for technical interviews.

## Question 1
**Topic:** Arrays
**Question:** Given an array of integers, find the maximum subarray sum (Kadane's algorithm). Explain your approach.
**Type:** Coding
**Difficulty:** Medium

## Question 2
**Topic:** Linked Lists
**Question:** How would you detect a cycle in a linked list? Implement the solution and explain the algorithm.
**Type:** Implementation
**Difficulty:** Medium

## Question 3
**Topic:** Trees
**Question:** Implement an in-order traversal of a binary tree both recursively and iteratively.
**Type:** Coding
**Difficulty:** Medium

## Question 4
**Topic:** Hash Tables
**Question:** Design a hash table from scratch. How would you handle collisions?
**Type:** Design
**Difficulty:** Hard

## Question 5
**Topic:** Sorting
**Question:** Explain the difference between stable and unstable sorting algorithms. Give examples of each.
**Type:** Concept
**Difficulty:** Easy

## Question 6
**Topic:** Dynamic Programming
**Question:** Solve the fibonacci sequence using dynamic programming. What's the time and space complexity?
**Type:** Implementation
**Difficulty:** Medium

## Question 7
**Topic:** Graphs
**Question:** Implement breadth-first search (BFS) and depth-first search (DFS) for a graph.
**Type:** Coding
**Difficulty:** Medium

## Question 8
**Topic:** Stacks and Queues
**Question:** How would you implement a queue using two stacks? Explain the time complexity of each operation.
**Type:** Implementation
**Difficulty:** Medium

## Question 9
**Topic:** Searching
**Question:** You have a rotated sorted array. How would you search for a target element efficiently?
**Type:** Problem Solving
**Difficulty:** Hard

## Question 10
**Topic:** Complexity Analysis
**Question:** Given a nested loop algorithm, how would you analyze its time complexity? Provide examples.
**Type:** Analysis
**Difficulty:** Easy
